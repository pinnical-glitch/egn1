# Design System - Home Energy Resilience Planner

## Overview
Technical, trustworthy, data-dense design for an engineering-portfolio-grade energy resilience dashboard.

## Style
- **Pattern**: Minimalism with technical precision
- **Color Palette**: Cool blues and grays with functional accent colors (solar=amber, battery=blue, load=red)
- **Typography**: Inter for body, JetBrains Mono for data/numbers

## Color System

### Primary Colors
- Primary: Deep blue (#3b82f6) for trust/technical feel
- Neutral: Cool grays for background and text

### Semantic Colors
- Success (Green #10b981): Positive/surplus energy
- Warning (Amber #f59e0b): Caution states
- Danger (Red #ef4444): Critical/depletion
- Info (Cyan #06b6d4): Informational

### Chart Colors
- Solar: Amber (#fbbf24)
- Battery: Blue (#3b82f6)
- Load: Red (#ef4444)
- Critical: Dark red (#dc2626)
- Important: Amber (#f59e0b)
- Optional: Gray (#6b7280)

## Typography
- **Font Family**: Inter (sans-serif), JetBrains Mono (monospace)
- **Scale**: 12px → 14px → 16px → 18px → 20px → 24px → 30px → 36px
- **Line Height**: 1.25 (tight), 1.5 (normal), 1.75 (relaxed)

## Spacing
- Base unit: 8dp (0.5rem)
- Scale: 4px, 8px, 12px, 16px, 20px, 24px, 32px, 40px, 48px, 64px, 80px

## Components
- **Cards**: White background, rounded-xl, shadow-md, subtle border
- **Buttons**: Primary (blue), Secondary (gray), Danger (red), Ghost (hover)
- **Inputs**: Clean borders, focus rings, error states
- **Badges**: Priority colors (Critical=red, Important=amber, Optional=gray)

## Responsive Breakpoints
- Mobile: < 640px
- Tablet: 640px - 1024px
- Desktop: > 1024px

## Accessibility
- Contrast ratio ≥ 4.5:1 for text
- Focus visible states
- Keyboard navigation support
- Screen reader labels
